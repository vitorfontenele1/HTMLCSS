- Links 

a = ancôra + hre = hypertext reference (link)
target(_blank) e rel(external) = abre duas páginas com o clique do link
rel(next) = diz ao mecanismo de busca que é a próxima página
rel(prev) = diz ao mecanismo de busca que o link refere-se a página principal
rel(nofollow) = diz ao mecanismo de busca que a página não é licitada por você 
target(_self) = diz ao mecanismo que é uma página própria
../ = para voltar muitas páginas atrás -- aprender linux]
