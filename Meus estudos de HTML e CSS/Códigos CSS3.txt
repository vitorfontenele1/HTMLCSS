Style = muda o estilo
 + background color = cor de fundo
 + fonte-family = fonte do site
 + fonte-size = tamanho do texto
 + color = cor do texto
 + text-align center = centraliza o texto
   - justify = alinha o texto
 + CSS link = CSS externo 
   - charset
 + Text_decoration = decora o texto
 + background-image = degrade 
   - podemos usar radia-gradiente para formar gradientes circulares
